## Default selected

This makes the checkbox item selected when initially opened. Use `defaultSelected` when not
controlling the checkbox item state.

<Example
	Component={DropdownItemCheckboxDefaultSelected}
	packageName="@atlaskit/dropdown-menu"
	featureFlagDefaults={{ 'platform-dst-motion-uplift-button': true }}
	featureFlagSwitchers={['motionPopup']}
/>

## Selected

This makes the checkbox item selected when `true`. Use `isSelected` when controlling the checkbox
item state.

<Example
	Component={DropdownItemCheckboxSelected}
	packageName="@atlaskit/dropdown-menu"
	featureFlagDefaults={{ 'platform-dst-motion-uplift-button': true }}
	featureFlagSwitchers={['motionPopup']}
/>
