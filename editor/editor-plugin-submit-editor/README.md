# Editor Plugin Submit Editor

Submit editor plugin for @atlaskit/editor-core

**Note:** This component is designed for internal Atlassian development.
External contributors will be able to use this component but will not be able to submit issues.

## Overview

The Submit Editor plugin enables submission of editor content with keyboard shortcuts. It integrates with the `@atlaskit/editor-core` to provide a configurable save/submit action and handles media upload states to prevent submission when uploads are in progress.

## Key features

- **Submit on shortcut** - Trigger submit action via keyboard shortcut
- **Media upload awareness** - Prevent submission when media uploads are in progress
- **Analytics tracking** - Track submit actions and document metrics
- **Optional Media plugin integration** - Works seamlessly with the Media plugin when available

## Usage
---
**Internal use only**

@atlaskit/editor-plugin-submit-editor is intended for internal use by the @atlaskit/editor-core and as a plugin dependency of the Editor within your product.

Direct use of this component is not supported.

Please see [Atlaskit - Editor plugin submit editor](https://atlaskit.atlassian.com/packages/editor/editor-plugin-submit-editor) for documentation and examples for this package.
