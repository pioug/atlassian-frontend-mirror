# @atlaskit/editor-element-browser

New UI for the QuickInsert menu and Right rail.

## Description

This package provides modern UI components for element insertion interfaces in the Atlassian Editor. It offers both quick insert panels for rapid content addition and side insert panels for more detailed element browsing and selection.

## Key Features

### Panel Components
- **QuickInsertPanel**: Fast content insertion interface with search capabilities
- **SideInsertPanel**: Comprehensive side panel for browsing and inserting elements

### UI Elements
- **IconButtonGroup**: Grouped icon-based navigation buttons
- **ListButtonGroup**: List-style button collections for element categories
- **ExpandablePanel**: Collapsible panels for organized content presentation
- **SearchPanel**: Search functionality for filtering available elements

### Interaction Features
- **Search and Filtering**: Fast search using Fuse.js for finding elements
- **Navigation**: Keyboard and mouse navigation support
- **Categorization**: Organized element grouping and display

## Usage

```typescript
import { QuickInsertPanel, SideInsertPanel } from '@atlaskit/editor-element-browser';
```

Detailed docs and example usage can be found
[here](https://atlaskit.atlassian.com/packages/editor/element-browser).
