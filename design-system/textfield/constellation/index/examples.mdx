## Basic

A basic text field. If you use a text field outside of a form component, always use a label and associate the label to the field properly.

<Example Component={TextFieldBasic} packageName="@atlaskit/textfield" />

## Text field in a form component

You'll often use text fields as a [form field](/components/form/examples#field), which must include a visible label.

<Example Component={TextFieldForm} packageName="@atlaskit/textfield" />

## Native validation

Validation can display a native error message related to the restrictions of the pattern attribute. Keep this text as short as possible. Use the [writing guidelines](/content/writing-style) for more help. For complicated infomation, provide a link to more information in a new browser tab.

<Example
  Component={TextFieldFormNativeValidation}
  packageName="@atlaskit/textfield"
/>

## Custom validation

Validation can display an error message related to the restrictions of the field. Keep this text as short as possible. Use the [writing guidelines](/content/writing-style) for more help. For complicated infomation, provide a link to more information in a new browser tab.

<Example
  Component={TextFieldFormValidation}
  packageName="@atlaskit/textfield"
/>

## Max characters

Text fields can be set with a maximum character limit for constraining the amount of content that can be entered. Inline validation can also be set to alert the user and clarify the request.

<Example Component={TextFieldMaxValue} packageName="@atlaskit/textfield" />
