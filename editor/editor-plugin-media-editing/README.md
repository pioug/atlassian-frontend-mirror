# Editor Plugin Media Editing

Media editing plugin for @atlaskit/editor-core

**Note:** This component is designed for internal Atlassian development.
External contributors will be able to use this component but will not be able to submit issues.

## Overview

The Media Editing plugin provides image editing capabilities for the Atlassian Editor. It integrates with the media plugin to enable users to crop, rotate, and flip images directly within the editor. The plugin includes a modal-based image editor with interactive controls for image transformation and manipulation.

## Key features

- **Image cropping** - Crop images with flexible selection and aspect ratio control
- **Image rotation** - Rotate images by 90-degree increments
- **Image flipping** - Flip images horizontally and vertically
- **Modal editor** - Interactive image editing interface with preview
- **Media integration** - Seamless integration with @atlaskit/editor-plugin-media
- **Editor state management** - Shared state for image editor visibility and selected media

## Usage
---
**Internal use only**

@atlaskit/editor-plugin-media-editing is intended for internal use by the @atlaskit/editor-core and as a plugin dependency of the Editor within your product.

Direct use of this component is not supported.

Please see [Atlaskit - Editor plugin media editing](https://atlaskit.atlassian.com/packages/editor/editor-plugin-media-editing) for documentation and examples for this package.
