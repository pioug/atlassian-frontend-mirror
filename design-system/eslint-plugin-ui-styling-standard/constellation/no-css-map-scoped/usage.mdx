# @atlaskit/design-system/no-css-map-scoped

Disallows usage of the experimental `cssMapScoped` API from `@compiled/react`. This API is internal
and is not part of the public Compiled CSS-in-JS interface.

This rule is maintained by another package and configured here; refer to
https://atlassian.design/components/eslint-plugin-design-system/no-css-map-scoped/usage for more
details.
