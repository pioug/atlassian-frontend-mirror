# TeamsAppInternalNavigation

Headless navigation utilities for consistent internal link behaviour across the Teams app.

## Usage

`import TeamsAppInternalNavigation from '@atlaskit/teams-app-internal-navigation';`

Detailed docs and example usage can be found [here](https://atlaskit.atlassian.com/packages/people-and-teams/teams-app-internal-navigation).
